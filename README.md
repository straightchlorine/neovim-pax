## Getting Started

### Prerequisites

* [Neovim](https://github.com/neovim/neovim) installed.

### Installation

Clone the repository into `~/.config/nvim`:

```sh
git clone https://github.com/straightchlorine/neovim-pax.git ~/.config/nvim
```

Then add LSPs and other tools with `:MasonInstall <tool>` to fit your needs.

## Alternatives

This is my private configuration. If you want something stable, these are recommended:

* [NvChad](https://github.com/NvChad/NvChad)
* [LazyVim](https://github.com/LazyVim/LazyVim)

Fast, well documented and maintained.
